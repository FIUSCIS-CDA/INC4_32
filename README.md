#### MODULE NAME
       INC4_32

#### DEPENDENCIES
       Adder_32, OneBitAdder

#### INPUTS
Name | Size(Bits)
-----|------------
A   |     32     

#### OUTPUTS
Name | Size(Bits)
-----|------------
S   |     32     

#### DESCRIPTION
       Computes  A  + 4, where A is a 32-bit value.  Stores result in a 32-bit
       value S.
